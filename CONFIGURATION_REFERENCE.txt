# Configuration Reference & Setup Guide

This file provides a consolidated reference of all required credentials, database setups, external accounts, environment variables, and the deployment readiness checklist.

---

## Required Accounts
1. **Google AI Studio**: Required to generate a Gemini API key for resume rewriting, parsing recommendations, and agent evaluations. Get one at: https://aistudio.google.com/app/apikey
2. **GitHub**: Required for version control and linking repositories to cloud hosting platforms.
3. **Railway**: Recommended for deploying the PostgreSQL database, ChromaDB service, and FastAPI backend.
4. **Vercel**: Recommended for deploying the static React + Vite frontend.

---

## Required Databases & Services
1. **PostgreSQL**: Used for storing persistent records like users, resumes, versions, ATS scores, match reports, and multi-agent workflow results.
2. **ChromaDB**: Used as the vector database for indexing and retrieving resume content during RAG chat queries.
3. **MLflow**: Used to log and track model metrics, run configurations, prompts, and evaluation outputs.
4. **Docker**: Used to run databases and run the application locally or in production containers.

## Deployment Readiness Checklist

Perform the following verification checks before deploying to staging/production:

- [ ] **API Keys Set**: Valid `GEMINI_API_KEY` configured.
- [ ] **Production Secret Key**: A unique 64-character hex string generated for `SECRET_KEY` (never leave as default).
- [ ] **Docker Env Binding**: Verified `docker-compose.yml` forwards the `GEMINI_API_KEY` environment variable.
- [ ] **CORS Origins**: Changed `allow_origins=["*"]` to your frontend production URL (e.g. Vercel domain) in `backend/app/main.py`.
- [ ] **Database Setup**: Database migrations run successfully using `alembic upgrade head` inside the container entrypoint.
- [ ] **ChromaDB Connection**: ChromaDB service is active and the host/port variables are configured correctly.
- [ ] **Vercel Routing**: `vercel.json` exists in the frontend folder with rewrite configurations.
- [ ] **Vite Base API**: Production backend domain set as the `VITE_API_URL` variable in Vercel config.
